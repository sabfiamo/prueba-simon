HOW TO PLAY?

  • Select the power button
  • Press start
  • The computer will give you a sequence of color and sound that you will have to replicate
  • If you fail, the computer will replicate the last sequence for you to try again
  • Strict mode will only allow you one miss, then restart the game
  • You have to replicate 10 rounds to win
